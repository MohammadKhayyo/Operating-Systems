String Parser
Authored by : Mohammad Khayyo


==Description==

parser string is implemented in the C language. You will receive from the user a sentence of one word or
More and print on the screen the number of words received, the words themselves and their length.

functions:
the program have 2 function:
1- printHowManyWords - the method will calculate how many words we entered through the string.
2- TheNumberOfLettersInEachWord- this method will cut the sentence into words and calculate how many letters are in them.


==Program Files==
ex1.c - they contain the main only and the functions.


==How to compile?==
compile: gcc ex1.c -o ex1
run: ./ex1

==Input:==
It was raining yesterday

==Output:==
4 words 
It : 2
was : 3
raining : 7
yesterday : 9
